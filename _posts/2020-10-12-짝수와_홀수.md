## 1. 짝수와 홀수
---
### 1.1 문제 설명
- 정수 num이 짝수일 경우 Even을 반환하고 홀수인 경우 Odd를 반환하는 함수, solution을 완성해주세요.

<br>

### 1.2 제한 조건
- num은 int 범위의 정수입니다.
- 0은 짝수입니다.

<br>

### 1.3 Solution

```python
def solution(num):
    if num % 2 == 0:
        return 'Even'
    else:
        return 'Odd'
```

<br>

```python
solution(2)
```
    'Even'
