# 두 수의 곱
[링크](https://school.programmers.co.kr/learn/courses/30/lessons/120804)

#### 문제 설명
- 정수 num1, num2가 매개변수 주어집니다. num1과 num2를 곱한 값을 return 하도록 solution 함수를 완성해주세요.

#### 제한사항
- 0 ≤ num1 ≤ 100
- 0 ≤ num2 ≤ 100

#### 문제 풀이
- Python은 `*`로 곱하기를 할 수 있다.


```python
def solution(num1, num2):
    answer = num1 * num2
    return answer
```
