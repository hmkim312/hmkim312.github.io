# 두 수의 차
[링크](https://school.programmers.co.kr/learn/courses/30/lessons/120803)

#### 문제 설명
- 정수 num1과 num2가 주어질 때, num1에서 num2를 뺀 값을 return하도록 soltuion 함수를 완성해주세요.

#### 제한사항
- 50000 ≤ num1 ≤ 50000
- 50000 ≤ num2 ≤ 50000

#### 문제 풀이
- `-`를 사용하여 두 수의 차를 구한다.


```python
def solution(num1, num2):
    return num1 - num2
```


```python
num1 = 2
num2 = 3
solution(num1, num2)
```
